Domogik 0.5.2 release

The Domogik team is proud to announce the release of Domogik 0.5.2!

This is to be used with Domoweb 0.5.2. You can use your current Domodroid release with it.

A new Android application is available : Butler Android. It is a voice interface application to speak with Domogik's Butler. This is a beta application : there is still a few bugs in it.

Domogik

You can read the release note to be aware of all the content of this release.

This release has only one major improvment : the Butler response time have been increased a lot. This improvment has a side effect : you will need to upgrade some packages to their latest release. Here are the concerned packages :

  • brain-base
  • brain-shopping
  • plugin-weather
  • plugin-callerid

Today, the best supported language by the Butler is French (just because the main Butler developper is French). If you want to help us improve the English part (or any other language), just come on irc and tell us : #domogik@freenode.net

Domoweb

Domoweb did not change a lot : only the health / body widget have been improved.

To conclude

This release goal was to fix all the little defaults of Domogik 0.5 and, thanks to the community, it is a success!
