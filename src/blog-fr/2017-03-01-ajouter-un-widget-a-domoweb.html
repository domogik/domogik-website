Ajouter un widget à Domoweb

Nous allons voir comment ajouter un widget à l'interface web Domoweb.

Domoweb est fournit avec plus de 30 widgets. Voici un aperçu des différents widgets existants :

Nous allons voir qu'ajouter un widget à une section (que nous pourrions aussi nommer une page, une vue, un dashboard) est très simple.

Prérequis

  • Avoir installé Domogik, Domoweb et créé au moins un device.

Ajouter un widget

Dans Domoweb, cliquez sur le logo de Domogik en bas à droite de l'écran.

Sélectionnez le menu Widgets.

Cliquez sur Add pour ajouter un nouveau widget.

Dans la liste des widgets, choisissez le widget adapté à votre besoin. Par exemple pour un capteur d'ouverture, allez sur le widget Sensor Bool et cliquez sur Add and close.

Le widget s'affiche et la page est encore en mode édition des widgets. Cliquez sur l'icône avec la roue crantée pour configurer le widget.

Dans le menu Sensors, sélectionnez le capteur (sensor) qui vous intéresse. Puis cliquez sur Save.

Le widget est configuré. Si vous souhaitez le déplacer, vous pouver utiliser les flèches en haut à droite du widget.

Recliquez sur le logo de Domogik en bas à droite, et choisissez Finish pour sortir du mode édition des widgets.

Et voilà, votre widget est affiché!

Vous pouvez modifier son style au besoin en retournant dans la configuration du widget.

Aller plus loin

Il est possible de configurer le style de chaque widget de manière indépendante (dans la configuration du widget) ou globale à la section (dans la configuration de la section). Les éléments suivants peuvent être personnalisés :

  • La couleur de fond : il peut s'agir d'un nom de couleur en anglais : red, blue, ..., d'un code hexadécimal RGB : #ff0000, #9ad45c, ..., ou encore du mot clé transparent pour avoir une couleur transparente.
  • La couleur du texte : elle se configure de la même manière que la couleur du fond.
  • La couleur des bords du widget : elle se configure de la même manière que la couleur du fond.
  • L'arrondi des angles du widget : la valeur peut être en pixels : 5px, 3px, 0px... ou en pourcentages : 5%, .... Une valeur de 50% va créer un widget rond! Il est possible de spécifier uniquement une valeur ou 4 valeurs séparées par un espace pour différencier chaque angle, par exemple : 10px 10px 0px 0px.
  • L'ombre du widget : il faut indiquer plusieurs valeurs : la distance horizontale, la distance verticale, le flou et pour finir la couleur en code hexadécimal RGB. Par exemple : 10px 10px 5px #656565.

Certains widgets ont des options spécifiques qui permettent de les personnaliser : il est donc intéressant de regarder toutes les options disponibles sur un widget!
