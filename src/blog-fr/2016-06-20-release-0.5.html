Sortie de Domogik 0.5

L'équipe de Domogik est fière de vous annoncer la sortie de la version 0.5 de Domogik. Cette version est accompagnée de Domoweb, l'interface de contrôle web en version 0.5 et de Domodroid en version 1.4.

Pour consulter la liste de toutes les évolutions, vous pouvez consulter la release note de cette version. Voici une liste simplifiées des évolutions :

Domogik

  • Fusion des composants suivants : interface d'administration et API Rest.
  • Simplification du processus d'installation (c'est un chantier qui continuera dans la prochaine version).
  • Réécriture du moteur de scénarios.
  • Ajout de nouvelles pages dans l'administration : timeline, suivi des batteries.
  • Ajout du support de plugins non xPL (pour simplifier la création des plugins).

Domoweb

  • Création de nouveaux widgets dans l'interface web.
  • Améliorations diverses sur les widgets qui existaient déjà.
  • Ajout d'une page pour parler avec le majordome.

Divers

  • Création d'une image docker. Il s'agit pour le moment d'une image orientée développeurs. Une image plus grand public (et plus légère donc) sera réalisée plus tard.
  • Mise en place du SSL : l'interface d'administration et l'interface web supportent désormais le SSL (non activé par défaut).

Concernant les packages, il y a pour le moment peu de packages sur le dépôt de packages. Nous avons pourtant de nombreux autres packages existant, mais ils doivent d'abord passer par une phase de validation.
