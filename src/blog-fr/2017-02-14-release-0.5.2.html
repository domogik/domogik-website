Sortie de Domogik 0.5.2

L'équipe de Domogik est fière de vous annoncer la sortie de la version 0.5.2 de Domogik.

Cette version est accompagnée de Domoweb, l'interface de contrôle web en version 0.5.2. Et bien entendu l'application Domodroid, interface de contrôle de Domogik sous Android, est toujours disponible.

Une nouvelle application Android débarque également : Butler Android. Il s'agit uniquement d'une interface vocale avec le Majordome. Nous en parlerons dans un billet dédié!

Domogik

Pour consulter la liste de toutes les évolutions de Domogik, vous pouvez consulter la release note de cette version.

Cette version contient une seule évolution majeure : l'amélioration du temps de réponse du Majordome aux requêtes. Cette amélioration a malheureusement un effet de bord et il faudra réinstaller la dernière version des packages suivants :

  • brain-base
  • brain-shopping
  • plugin-weather
  • plugin-callerid

Par extension, le package brain-base apporte beaucoup de confort dans l'utilisation du Majordome avec une meilleure intégration de formules usuelles de langage naturel. Par exemple, les phrases suivantes sont équivalentes :

— Quelle est la température du salon ?

— Peux-tu m'indiquer la température du salon ?

— Est-ce que tu peux me dire quelle est la température dans le salon ?

— Il fait combien dans le salon ?

— Combien fat-il dans le salon ?

— Pourais-tu me donner la température dans le salon ?

Domoweb

Domoweb n'a quasiment pas évolué, il y a simplement eu un correctif sur le widget health / body.

Le mot de la fin

Cette version qui sort deux mois après la version précédente a permit, grâce à une communauté active de corriger les petites imperfections de la version 0.5.

Nous vous donnons rendez-vous dans quelques semaines pour la prochaine version, la 0.6 qui commencera à introduire de nouvelles possibilités plus qu'intéressantes!
